# Contributing

## Welcome contributions

- Word bank improvements (`On`, `Ologn`, `O1`)
- Bug reports and reproducible fixes
- Anti-cheat threshold tuning and validation
- Performance improvements on typing/render paths

## Please avoid

- Adding new runtime dependencies without approval
- Replacing the terminal-first UI style with component frameworks
- Refactoring unrelated files in the same PR

## PR checklist

- Tested on Chrome and Firefox
- `npm run lint` and `npm run build` pass
- No new dependencies unless explicitly justified
- For word bank PRs, examples are realistic and syntactically valid
